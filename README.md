# Time-Series-Forecasting-Adv-Research-
Project Overview This project forecasts German electricity load using weekly and hourly data.  
The project compares:
- Benchmark forecasting models
- SARIMA
- SARIMAX with temperature as an exogenous variable
- Random Forest Regressor
- LSTM Neural Network
## Dataset
- German electricity load (2015–2020)
- Berlin temperature from Open-Meteo Archive API
